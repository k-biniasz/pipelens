# TiDB Meeting Intelligence

An AI-powered sales intelligence tool built for PingCAP's go-to-market team. Upload a spreadsheet of marketing-sourced meetings and get instant AI analysis across conversion patterns, ICP definition, LinkedIn targeting, pipeline health, AE performance coaching, and account re-engagement opportunities.

Built with React, powered by Claude (Anthropic).

---

## What It Does

Upload any `.xlsx` or `.csv` export from your CRM or marketing automation platform. The tool auto-detects columns and provides nine analysis modules:

| Tab | What it analyzes |
|-----|-----------------|
| **Overview** | Conversion rates, top signals, dimension breakdowns |
| **Conversion Analysis** | ICP definition from converted leads |
| **Notes Analysis** | AI synthesis of IQM, SDR, and AE notes |
| **LinkedIn Intelligence** | Targeting segments, keywords, messaging angles |
| **Disqualified** | Bad-fit themes and disqualification patterns |
| **Stuck in Pipeline** | Gap analysis vs converted leads |
| **AE Performance** | Individual and team coaching analysis |
| **Who NOT to Target** | Negative ICP, persona exclusions, LinkedIn negative keywords |
| **Re-engage Accounts** | Accounts worth re-targeting despite non-conversion |

All analysis is pre-configured with TiDB/PingCAP context so output is product-specific, not generic.

## Deploying to Vercel (Recommended)

This is the fastest path to sharing the tool with your team. No servers, no infrastructure.

### 1. Fork or clone this repo

```bash
git clone https://github.com/YOUR_ORG/tidb-meeting-intel.git
cd tidb-meeting-intel
```

### 2. Push to GitHub

If you cloned it, create a new repo on GitHub and push:

```bash
git remote set-url origin https://github.com/YOUR_ORG/tidb-meeting-intel.git
git push -u origin main
```

### 3. Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **Add New Project** → select this repo
3. Vercel will auto-detect it as a Vite project
4. Before deploying, add your environment variable:
   - Key: `ANTHROPIC_API_KEY`
   - Value: your key from [console.anthropic.com](https://console.anthropic.com/)
5. Click **Deploy**

Your team gets a URL like `https://tidb-meeting-intel.vercel.app`. Share it and you're done.

> **Security note**: The API key lives only in Vercel's environment — never in the browser or the repo. The `/api/claude` serverless function acts as a proxy.

## Running Locally

```bash
npm install
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY to .env.local

# Terminal 1 — Vite dev server
npm run dev

# Terminal 2 — Vercel serverless functions locally
npx vercel dev
```

Open `http://localhost:5173`.

> You need the [Vercel CLI](https://vercel.com/docs/cli) installed (`npm i -g vercel`) for the API proxy to work locally. Alternatively, run `vercel dev` on its own — it starts both the frontend and the function.

## Column Detection

The tool auto-detects columns from your spreadsheet headers. Supported field names:

| Field | Detected column names |
|-------|-----------------------|
| Converted / Won | `converted`, `is_converted`, `opportunity`, `won` |
| Lead Status | `lead_status`, `status` |
| AE Owner | `ae`, `owner`, `rep`, `sales rep`, `account executive` |
| IQM Score | `iqm_score`, `iqm quality score`, `meeting score` |
| AE Score | `ae_score`, `ae feedback score`, `ae rating` |
| Account Name | `account_name`, `company`, `organization` |
| LinkedIn fields | `linkedin_url`, `linkedin_summary`, `linkedin_skills`, etc. |

If your column name isn't detected, you can manually select the conversion column from a dropdown after upload.

## Project Summary

> *The following summary was generated by Claude (Anthropic) based on the project.*

TiDB Meeting Intelligence is an internal AI tool built as part of PingCAP's company-wide AI initiative. It transforms raw marketing meeting data into structured sales intelligence without requiring any data science expertise or manual analysis.

The core problem it solves: marketing teams generate hundreds of meeting records — notes, LinkedIn profiles, lead attributes, conversion outcomes — but extracting actionable patterns from that data traditionally requires hours of manual review or dedicated analytics resources. This tool automates that entirely.

A sales or marketing team member uploads their meeting export, and within seconds gets AI-generated answers to the questions that matter most: Who is our ideal customer? What do people say in meetings before they become opportunities? Which accounts should we re-target? Which personas should we stop targeting? How are individual AEs performing against the team average?

The tool is built to be TiDB-specific — all analysis is framed around PingCAP's positioning, competitive landscape, and ICP, so output can be used directly in campaign briefs, SDR playbooks, and AE coaching sessions without additional editing.
